# Vora UI

Open source crypto market analytics components, distributed shadcn-style.
Install the source into your project and own it.

```bash
npx shadcn@latest add @voraui/trading-chart
```

## Components (v1)

- Fear & Greed gauge
- Altseason index gauge
- BTC Rainbow chart
- Trading chart with trade markers (flagship)

Every component renders live data out of the box from free public APIs (alternative.me, CoinGecko, Binance), and accepts your own data via props.
